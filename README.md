# Kanha Poshak Bhandar Website

A modern, responsive React application for Kanha Poshak Bhandar, located in Badshahpur Main Bazar, Gurugram.

## Features
- **Beautiful Design**: Premium aesthetic with divine color palette (Saffron, Gold, Maroon).
- **Product Showcase**: Display Poshak, Accessories, and Idol dresses.
- **Admin Panel**: Built-in feature for shop owner to add new items and prices.
- **Sample Data**: Pre-loaded with generated high-quality images of Kanha Poshak and accessories.

## How to Run

Since Node.js was not detected in the current environment, you will need to install it to run this application.

1.  **Install Node.js**: Download and install from [nodejs.org](https://nodejs.org/).
2.  **Open Terminal**: Open a command prompt or terminal in this folder.
3.  **Install Dependencies**:
    ```bash
    npm install
    ```
4.  **Start Development Server**:
    ```bash
    npm run dev
    ```
5.  **Open in Browser**: Click on the link shown in terminal (usually http://localhost:5173).

## Project Structure
- `src/components`: specialized components for Header, Hero, products, and Admin.
- `src/data.js`: Initial product list.
- `public/images`: Generated assets.
- `src/index.css`: Global styles and variables.
